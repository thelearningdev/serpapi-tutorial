02 / formula

Scoring for Pre-Viral Potential

score = (avg_rating × 10) − log(review_count + 1) + trend_multiplier
avg_rating × 10
Rewards quality — 4.8 beats 4.3
− log(review_count + 1)
Penalizes popularity — 2000 reviews loses more than 50
+ trend_multiplier
Adds back for recency signals from the SerpAPI response

04 / design decisions

Why Not Other Approaches?

Simple ratio
rating / reviews
Divide rating by review count to penalize popular places
A place with 1 review and 5.0★ scores highest. Pure noise.
Bayesian avg
prior + observations
Smooth ratings toward a global mean using a prior
Pulls low-volume places toward the mean — buries the gems we're looking for.
Wilson score
confidence interval
Lower bound of confidence interval on positive rate
Designed for binary thumbs up/down — not a 5-star scale.
Log dampener
our approach
Grows fast at low counts, flattens at high volume. Intuitive, tunable, and maps directly to SerpAPI fields.

07 / scorer

The Scoring Function

score = (rating × 10) − log(review_count + 1) + trend_multiplier
scorer.py
import math def compute_score(place: dict) -> float: rating = place.get("rating", 0) or 0 review_count = place.get("reviews", 0) or 0 trend = _trend_multiplier(place) return round( (rating * 10) - math.log(review_count + 1) + trend, 2 )
07 / scorer

The Trend Multiplier

+2.0
"Newly opened" in the hours field Strongest signal — early ratings on a brand-new place
+1.0
"Popular" label in extensions Google's own trending flag — surfaces in the API response
+0.5
High photo-to-review ratio People are visiting and photographing before they write reviews
+0.5
Budget price ($ or $$) with 4.5+ rating Hidden gems are disproportionately cheap
07 / scorer

The Trend Multiplier — Implementation

SerpAPI fields → trend score

scorer.py
def _trend_multiplier(place: dict) -> float: score = 0.0 review_count = place.get("reviews", 0) or 0 if 10 <= review_count <= 100: # early traction sweet spot score += 5.0 elif review_count < 10: # too new — lower confidence score += 2.0 if place.get("owner_answer"): # active owner = growing business score += 3.0 if place.get("hours") and "Opens" in str(place.get("hours", "")): score += 1.0 # recently claimed listing if not place.get("price"): score -= 1.0 # incomplete listing — penalize return score
